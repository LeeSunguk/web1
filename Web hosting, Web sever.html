Web hosting, Web sever

Web hosting, Web sever 중 Web hosting은 자신이 만든 웹을 다른 회사의 서버를 통해 웹을 여는 것이고
Web sever를 사용하려면 두 대의 컴퓨터 즉, client 컴퓨터와 sever 컴퓨터가 필요하다. 여기서 client 컴퓨터가 sever 컴퓨터에게 요청을 하면 sever 컴퓨터가 응답을 하는 방식으로 이루어진다. Web sever를 직접 여는 것은 Web hosting에 비해 복잡하고 어렵고 client 컴퓨터와 sever 컴퓨터 두 대가 필요하다는 단점이 있다.
